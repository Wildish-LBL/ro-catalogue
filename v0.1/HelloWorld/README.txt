Hello World Research Object

This is a simple Research Object that embeds a simple "Hello World" workflow.

To run this workflow in Taverna, first edit the fileurl_value and outputFile_value to reflect paths on the local system to the Research Object directory.

Research Object files are:

README.txt - this file
make.sh - sample script to create Research Object structure using RO Manager
TavernaHelloWorld.t2flow - Taverna "Hello world" workflow
InputName.txt - sample input file
HelloOutput.txt - result from running workflow with sample input file
HelloWorld-wfdesc.rdf - Workflow description using WFDesc vocabulary terms (http://wf4ever.github.com/ro/)
